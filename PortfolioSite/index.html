What Do Web Devolopers do?

Web developers in short, create web applications. We write the code for the pages, the style to a website, and the scripts that add user interactivity.

Front-End

User Interface layer usually consisting of a mix of HTML, Javascript, CSS, Flash, and various server-side code like ASP.NET. Think of this as being closest to the user in terms of code.

Middle Tier

Middle-tier -> One tier back, generally referred to as the "plumbing" part of a system. JavaScript and C# are common languages for writing this part that could be viewed as the glue between the UI and the data.

Back-End

Databases and other data stores are generally at this level. Oracle, MS-SQL, MySQL, SAP, and various off-the-shelf pieces of software come to mind for this piece of software that is the final processing of the data.

Full Stack

So what is Full Stack? A full stack developer uses all 3 of the tiers of development to create complete websites.
